Banking Simulation Project

This project is a set of JavaScript exercises that simulate different banking operations.  
Each part is labeled as a "Question" and shows how to solve problems with code.



Question 1: ATM & Transactions
. Start with a balance of 1000
. Deposit and withdraw money
. Undo the last transaction
. Apply a $10 maintenance fee
. Keep a history of the last 10 transactions



Question 2: Loan App
. Work with credit scores
. Find eligible scores (above 700)
. Adjust scores (+20)
. Calculate total risk
. Check if any score is perfect (900)
. Find risky scores below 500



 Question 3: Fraud Detection
. Detect a fraud transaction in a list
. Show the index of fraud
. Display the last 3 transactions
. Remove the fraud entry
. Print cleared transactions


 Question 4: Bank Merge
.Combine customer lists from two branches
.Flatten messy nested data
.Sort names in reverse order (Z–A)
.Create a banner string
.Show teller windows status



Question 5: Security & Report
.Validate passwords (must be strong and contain vowels)
.Generate yearly reports:
.Special audit years (leap years)
.5‑year anniversaries
.10‑year anniversaries
